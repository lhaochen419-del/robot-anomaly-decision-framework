# Repository manifest

- raw_data/: no third-party data redistributed; contains README.
- processed_data/: no derivative third-party data redistributed; contains README.
- dataset_acquisition/: official links, placement instructions and expected tree.
- configs/: copied YAML configuration files.
- command_manifests/: command and output manifests.
- scripts/: public-release code.
- results_aggregated/: aggregate benchmark and strategy outputs.
- supplementary_tables/: supplementary statistical/result tables.
- figures/: figure PDFs/PNGs/SVGs and source maps.
- environment/: requirements and environment notes.
