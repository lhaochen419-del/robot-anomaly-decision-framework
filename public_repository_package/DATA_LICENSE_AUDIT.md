# Data License Audit

- IMAD-DS: official source https://zenodo.org/records/12665499, DOI https://doi.org/10.5281/zenodo.12665499. Local copies exist, but redistribution rights were not verified in the available artifacts. Raw and processed dataset files are therefore not copied into this public package.
- RoAD: official source https://gitlab.com/AlessioMascolini/roaddataset and paper DOI https://doi.org/10.1109/IECON51785.2023.10311726. Redistribution rights were not verified in the available artifacts. Dataset content is not copied.
- NIST UR: official source https://doi.org/10.18434/M31962. Used only for readiness/exclusion context, not primary binary evidence.
- KUKA: official source https://zenodo.org/records/6461868. Used only for readiness/exclusion context, not primary binary evidence.

Decision: third-party raw and processed data are not redistributed in this package unless the authors later confirm compatible redistribution rights. The package instead provides official download links, expected file trees, checksums where available, adapter scripts and data-placement templates.
